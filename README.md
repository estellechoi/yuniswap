# Yuniswap

Clone contracts of Uniswap to understand how DEX works

<br />

### Install Solidity compiler

```zsh
brew tap ethereum/ethereum
brew install solidity
```

```zsh
# Try solc command (solc is solidity compiler)
solc --verison
```

<br />

### Install packages

```zsh
# npm install -g yarn (if needed)
yarn install
```
